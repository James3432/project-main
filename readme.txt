The source files are split into 3 sections:
  
  Reconstructor:  The primary body of code for reconstructing models from Kinect data
  KinectGrabber:  C++ code for retrieving data from the Kinect
     - RobotController:  C# code for controlling a Roomba iRobot, interfaces with KinectGrabber
